# go-fisio-concurrencia

## Descripción del Proyecto

Este proyecto es una implementación en Go de un sistema de simulación de fisioterapia con concurrencia. Fue desarrollado como una forma de profundizar en los conceptos de programación concurrente y distribuida, utilizando un lenguaje moderno y robusto como Go. La idea surgió de la necesidad de aplicar conocimientos teóricos a un problema práctico, inspirándose en las clases de programación concurrente de la universidad y el enfoque de "aprendizaje seguro" que ofrece Go.

## Motivación Personal

La motivación principal detrás de este proyecto fue aprender Go, un lenguaje que considero un "safe place" por su modernidad y solidez. Las clases de Gentleman Programming y la conexión con mis estudios universitarios en programación concurrente y distribuida me impulsaron a traducir un ejercicio práctico a este nuevo entorno. Mi objetivo era lograr una comprensión más profunda y asimilación de los conceptos a través de la implementación.

## El Desafío

El mayor desafío fue el "cara a cara" inicial con las goroutines y los canales de Go. Aunque la lógica subyacente a la concurrencia ya la entendía, la magnitud de las implicaciones y las sutilezas de su implementación en Go fueron un reto considerable. Fue una experiencia intensa que me obligó a pensar diferente sobre cómo se manejan los procesos concurrentes.

## Aprendizajes Clave

Este proyecto fue una experiencia de aprendizaje inmensa. Me permitió reforzar y asimilar de manera práctica conceptos fundamentales como hilos (threads), punteros y la gestión de la concurrencia. La implementación me ayudó a solidificar la teoría y a ver cómo estos conceptos se aplican en un entorno real.

## Uso del Proyecto

Este proyecto fue concebido meramente como una herramienta personal de aprendizaje de Go. No está diseñado para ser una aplicación de producción ni para recibir contribuciones activas en su estado actual, a menos que alguien lo encuentre útil para sus propios fines de estudio o experimentación.

## Primeros Pasos

Para ejecutar este proyecto en tu máquina local, sigue estos pasos:

### Prerrequisitos

Asegúrate de tener Go instalado. Puedes descargarlo desde [golang.org](https://golang.org/dl/).

### Instalación

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/Gabo-Dev/go-fisio-concurrencia.git
    ```
2.  Navega al directorio del proyecto:
    ```bash
    cd go-fisio-concurrencia
    ```

### Ejecución

Para ejecutar la simulación:

```bash
go run main.go
```

Esto iniciará la simulación y mostrará la salida en la consola.
